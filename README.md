# Umareal Tournament

Tournament management app for Umamusume club with triple-threat racing format.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Run in development mode:
```bash
npm run electron:dev
```

## Features

- Always-on-top window for streaming
- Tournament bracket management for 1v1v1 races
- Real-time score tracking

## Tech Stack

- Electron
- Vue 3
- Vuetify 3
- Vite
